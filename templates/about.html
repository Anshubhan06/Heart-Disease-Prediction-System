About the project

AI Healthcare — Version 1.0

A web application combining machine learning, clinical data and thoughtful design to make healthcare insights more accessible.

About AI Healthcare

AI Healthcare is a web application that uses Machine Learning to estimate the risk of heart disease using clinical parameters. It demonstrates how AI can assist healthcare professionals by providing a quick preliminary assessment.

Project Objective

The goal of this project is to build a complete AI Healthcare web application using Machine Learning, Flask and a professional frontend rather than only creating a prediction script.

Machine Learning Model

The prediction engine uses a Random Forest Classifier trained using Scikit-learn. Random Forest provides good accuracy, reduces overfitting and supports predict_proba() which is used to calculate the confidence score.

Dataset

The model is trained using the Cleveland Heart Disease Dataset from the UCI Machine Learning Repository. The dataset contains 303 patient records with 13 clinical attributes such as age, cholesterol, blood pressure, ECG results and more.

Technologies Used

Python Flask Scikit-learn Random Forest Pandas NumPy HTML CSS JavaScript Git GitHub

Future Scope

Future versions may include Diabetes Prediction, BMI based analysis, integration with wearable devices, multilingual support and cloud deployment for real-world healthcare applications.
