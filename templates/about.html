About the project

AI Healthcare — Version 1.0

A web application combining machine learning, clinical data, and thoughtful design to make healthcare insights more accessible.

About AI Healthcare

AI Healthcare is a web application that uses machine learning to estimate the risk of heart disease based on clinical parameters. It was built to demonstrate how AI can assist — not replace — healthcare professionals by providing a fast, accessible first-step analysis for patients and students alike.

Project objective

The goal of this project is to build a real, deployable AI healthcare platform — not just a prediction script. It combines machine learning, backend development, and frontend design into a cohesive product that feels professional and is easy to use by anyone, with no medical background required.

Machine learning model

The prediction engine uses a Random Forest Classifier trained using Scikit-learn. Random Forest was chosen because it handles both categorical and numerical data well, is robust against overfitting, and provides probability scores via predict_proba() — enabling the confidence score shown on the dashboard. The model achieves approximately 79.6% accuracy on the test set.

Dataset

The model was trained on the Cleveland Heart Disease Dataset from the UCI Machine Learning Repository — one of the most widely cited datasets in medical ML research. It contains 303 patient records with 13 clinical attributes including age, blood pressure, cholesterol, ECG results, and more. The target variable indicates the presence or absence of heart disease.

Technologies used

Python Flask Scikit-learn Random Forest Pandas NumPy HTML / CSS JavaScript Git & GitHub

Future scope

Version 1 focuses on heart disease risk prediction. Future versions may include additional AI modules (diabetes risk, BMI-linked analysis), integration with wearable health data, multi-language support, and a more advanced dashboard with trend tracking over time.
